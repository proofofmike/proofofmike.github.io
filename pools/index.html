Solo Bitcoin Mining Pools & Setups Compared

Solo mining is a lottery. One honest block can change your life — and the pool or setup you pick actually matters. This is my straight-up comparison of the solo options that real hobby miners are using today, based on firsthand experience instead of marketing.

Focused on Bitcoin solo mining Address-in-template transparency noted where relevant No cloud mining, no sketchy “lifetime contracts”

1. CKPool — The OG Solo Pool

solo.ckpool.org

  • Status: Longest-running dedicated solo pool
  • Fee: 2%
  • Address in block template: ❌ No
  • Setup difficulty: Very easy
  • Operator: “Con”, running this since the early days

I’ve used CKPool off and on since around 2015. I even hit two blocks on CKPool back when they were worth peanuts — and just to be clear so nobody imagines I’m quietly sitting on millions, I don’t still have those coins. Life was messy; they’re gone.

Pros

  • Dead-simple configuration
  • Longest-running solo pool with real history
  • Solid uptime and reliability
  • Great beginner solo pool

Cons

  • Your payout address is not in the block template
  • Single-person operation
  • 2% fee
  • Limited advanced features

2. Ocean + DACTUM — Modern, Transparent, High-Performance

ocean.xyz · dactum.io

  • Status: Newer, cutting-edge tech stack
  • Fee: 1%
  • Address in block template: ✔️ Yes
  • Setup difficulty: Medium

Ocean handles block building and propagation. DACTUM provides the miner UI, dashboard, and node integrations. You typically pair this with a full node: Bitcoin Core or Bitcoin Knots.

Pros

  • Your address in the block template → trust-minimized payouts
  • Excellent propagation
  • Actively developed by decentralization-focused engineers
  • Clean dashboard
  • Supports split mining

Cons

  • Requires a node + some Linux comfort
  • More moving parts than a hosted pool

3. Umbrel Solo Mining App — Zero-Fee, Point-and-Click

umbrel.com

  • Status: Node-in-a-box with a friendly UI
  • Fee: 0%
  • Address in block template: ✔️ Yes
  • Setup difficulty: Easy

Umbrel wraps a Bitcoin node and mining stack into a point-and-click environment while still running your own full node.

Pros

  • Zero fee — you keep 100% if you hit a block
  • You control the node & block template
  • Easier than managing a raw Linux stack
  • Nice UI for monitoring

Cons

  • Requires an Umbrel device or compatible server
  • Less flexible than a full custom stack
  • You still must monitor your node

4. Fully Self-Hosted Solo Mining (DIY)

  • Status: Most sovereign option
  • Fee: 0%
  • Address in block template: ✔️ Yes
  • Setup difficulty: Advanced

Fully self-hosted solo mining means running your own node, mempool, block templates, and stratum proxy. Maximum sovereignty — maximum responsibility.

Pros

  • Fully trustless
  • Zero fees
  • Maximum control
  • Ideal for purists

Cons

  • Requires deep Bitcoin + Linux knowledge
  • You handle uptime, networking, propagation
  • Easy to misconfigure without caution

Other Pools & Large Centralized Options

There are big mining pools with “solo modes” (ViaBTC and others), plus gated or closed pools and smaller regional setups.

I’m not listing or reviewing them here for a simple reason: I don’t personally mine on centralized “solo modes,” and I generally don’t recommend them for true solo setups unless you trust the operator and understand the tradeoffs.

CKPool is the one exception I’m comfortable suggesting because it’s been around for a decade, has a long public track record, and I’ve personally used it on and off since around 2015. For everything else, I stick to setups where the block template is genuinely yours.
