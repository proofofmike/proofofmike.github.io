# ProofOfMike.github.io

This repository hosts the Proof Of Mike site and vendor subpages.

- `/` redirects to `/tch/` (TinyChipHub partner page)
- `/tch/` is the main affiliate page for TinyChipHub, with promo code **PROOFOFMIKE** and your store link.
- Add more vendors later as `/vendor/` folders, each with its own `index.html`.

### Local dev
Open `tch/index.html` in a browser, edit files, then push:

```bash
git add .
git commit -m "update site"
git push origin main
```
